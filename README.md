
# AutoMerge AI

Data Collection, And Exploratory Data Analysis

## Setup

Clone the project
```zsh
  git clone https://github.com/bishwash-007/automergeai.git
  cd automergeai
```
Setup Environment

```zsh
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
```
